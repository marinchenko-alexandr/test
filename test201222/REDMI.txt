Тест проверяет корректную работу сократителя длины ссылок

Для запуска теста необходимо установить следующие компаоненты: 


1. Установить Python от версии 3.9 - https://www.python.org/

2. Установить Selenium  (pip install selenium) или с помощью IDE 

3. Загрузите Selenium WebDriver с https://chromedriver.chromium.org/downloads (выберите версию, совместимую с вашим браузером) и добавте в папку с проектом
